# 个人作品集网站

一个现代化、响应式的个人作品集网站，展示你的项目、技能和联系方式。

## 功能特点

- ✨ **现代化设计** - 采用渐变色彩和流畅动画效果
- 📱 **完全响应式** - 适配桌面、平板和移动设备
- 🎨 **美观的UI** - 精心设计的用户界面和交互体验
- ⚡ **流畅动画** - 滚动动画和悬停效果
- 🚀 **性能优化** - 轻量级代码，快速加载

## 项目结构

```
个人作品集/
│
├── index.html      # 主HTML文件
├── styles.css      # 样式文件
├── script.js       # JavaScript交互文件
└── README.md       # 项目说明文档
```

## 使用说明

1. **打开网站**
   - 直接在浏览器中打开 `index.html` 文件即可查看网站

2. **自定义内容**
   - 编辑 `index.html` 文件，修改以下内容：
     - 导航栏中的品牌名称
     - 英雄区的姓名和描述
     - 关于我部分的个人介绍
     - 技能卡片中的技能描述
     - 作品展示区的项目信息
     - 联系方式中的邮箱和社交媒体链接

3. **自定义样式**
   - 编辑 `styles.css` 文件中的 CSS 变量来更改主题颜色：
     ```css
     :root {
         --primary-color: #6366f1;    /* 主色调 */
         --secondary-color: #8b5cf6;  /* 次要色调 */
         /* 更多颜色变量... */
     }
     ```

4. **添加项目图片**
   - 在 `project-image` div 中添加实际的项目截图
   - 或者使用 CSS 背景图片来替换当前的渐变背景

## 主要部分

### 1. 导航栏
- 固定顶部导航
- 移动端响应式菜单
- 平滑滚动到各个部分

### 2. 英雄区
- 醒目的标题和介绍
- 行动按钮（查看作品、联系我）
- 装饰性浮动形状动画

### 3. 关于我
- 个人介绍
- 统计数据展示（项目数、经验、满意度）

### 4. 技能展示
- 技能卡片网格布局
- 悬停效果

### 5. 作品展示
- 项目卡片网格
- 悬停显示项目链接
- 技术标签

### 6. 联系方式
- 联系信息展示
- 联系表单（需要后端支持才能实际发送邮件）
